This repository contains multiple independent Java projects that demonstrate strong understanding of:

- Data Structures (Linked Lists, Trees, Arrays)

Each project is implemented separately and focuses on a specific concept.

---

## 📁 Projects Included

### 🔹 HasCommonElementSorted
Checks whether two sorted structures share a common element efficiently.

### 🔹 SortedDoubleLinkedListCleaner
Implements a sorted doubly linked list with cleanup and maintenance operations.

### 🔹 BinaryTreePairsSum
Finds pairs in a binary tree that sum to a target value using traversal techniques.

### 🔹 ScheduleManager
A simple scheduling system demonstrating OOP design and structured data handling.
